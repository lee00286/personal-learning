# personal-learning

This is a collection of my personal learning projects that tackle different topics. Each project took ~1 hour to complete.

## 001-investigate-data-fetchng-strategy

This project investigates the difference between SSR, CSR, SSG, and ISR.

Please refer to the [README](001-investigate-data-fetchng-strategy/README.md) for more details.

## 002-investigate-event-loop

This project investigates how Node.js handles concurrent requests using a single-threaded event loop and non-blocking I/O.

Please refer to the [README](002-investigate-event-loop/README.md) for more details.

## 003-investigate-hash-table

This project investigates the internal mechanics of data structures like `Object` and `Map` in JavaScript and manage data efficiently using hash functions. It also compares hash algorithms and implement a collision-resistant data structure.

Please refer to the [README](003-investigate-hash-table/README.md) for more details.

## 004-investigate-rsc-client-component

This project investigates the optimization of bundle size with Server Components (RSC) and component composition. It also analyzes the impact of using RSC and Client Components on the performance of Next.js applications.

Please refer to the [README](004-investigate-rsc-client-component/README.md) for more details.
